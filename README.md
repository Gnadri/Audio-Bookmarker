# Nebula Dashboard Extension

A premium Chrome Extension featuring a high-end popup and a built-in interactive dashboard.

## Features
- **Modern UI**: Built with a sleek, dark-mode aesthetic using glassmorphism and smooth animations.
- **Internal Webpage**: A full-screen dashboard accessible via the extension popup or options page.
- **Chrome V3**: Uses the latest Manifest version for performance and security.

## How to Install
1. Open Google Chrome.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right corner).
4. Click **Load unpacked**.
5. Select the folder: `C:\Users\Lenovo\.gemini\antigravity\scratch\chrome-extension-webpage`
6. Pin the extension to your toolbar to see the Nebula icon!

## How to use
- Click the extension icon to see the **Popup**.
- Click **Open Dashboard** in the popup to view the full internal webpage.
